# ✨ ART RARE DEX — Pokédex TCG Project

Welcome to **ART RARE DEX**, a premium digital gallery and interactive Pokédex. 

> "A visual journey to dedicate at least one card to each of the 1,025 Pokémon — yes, even those that aren't loved enough."

---

## 🎨 The Project
This is a personal artistic challenge to create custom TCG (Trading Card Game) illustrations for every single Pokémon species in existence, from Kanto to Paldea. 

### 🤖 Artistic Process & Transparency
Every card in this collection features unique illustrations generated and refined using **AI-assisted tools**. 
* **Curation:** Each output is manually selected and edited to fit the "Art Rare" aesthetic.
* **Design:** The web interface and card layouts are custom-built to provide a "Masterpiece" feel for every Pokémon, especially the underrated ones.

---

## 📊 Collection Progress
Currently, the Pokédex is a living archive that updates as new designs are finished.

- **Status:** 18 / 1,025 Illustrated 

---

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Gold-themed UI), and Vanilla JavaScript.
* **Data:** Powered by PokeAPI for sprites and national dex ordering.
* **Hosting:** Live on [GitHub Pages](https://tzfrann.github.io/ArtRareDex/).

---

## 🚀 How to Explore
1. Visit the [Live Gallery](https://tzfrann.github.io/ArtRareDex/).
2. Use the **"Show Gallery"** button to see only the completed illustrations (like **Abra**).
3. Click on any card to open the High-Resolution modal and explore variants.
